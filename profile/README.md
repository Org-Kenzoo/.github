# Kenzoo

**Mets en relation clients et prestataires de services en Côte d'Ivoire.**

Kenzoo est une plateforme qui simplifie l'accès aux services du quotidien : trouver un prestataire, demander un devis, payer en sécurité, et gagner des Gombos à chaque transaction.

## Fonctionnalités clés

- **Explorer** — Recherche de prestataires par catégorie, carte ou grille
- **Ken Express** — Service rapide dans un rayon de 2 km
- **Devis & Escrow** — Paiement sécurisé libéré à la validation de la prestation
- **Messagerie & Appels** — Chat temps réel et appels vidéo intégrés
- **Gombos** — Système de points de fidélité propre à Kenzoo
- **Boost** — Mise en avant des prestataires sur la carte

## Stack technique

```
Mobile     →  React Native 0.81 · Expo SDK 54
Backend    →  Supabase (Postgres · Auth · Realtime · Storage · Edge Functions)
Web        →  React 19 · Vite
Paiements  →  Escrow intégré · Portefeuille Kenzoo
```
